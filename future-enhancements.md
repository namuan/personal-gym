# Future Enhancements & Expansion Roadmap

*This document captures ideas and plans beyond the MVP and post-MVP (Phase 2) scope. These are not committed but are kept on record for future consideration.*

---

## Phase 2 — Enhancement (post-MVP)

| Feature | Description |
|---|---|
| **Multiple named workout plans** | Allow users to create, name, and switch between several workout plans (e.g., "Upper Body", "Leg Day", "Full Body"). |
| **Exercise library** | Save exercises as reusable entries that can be quickly added to any plan without retyping. |
| **Static images for guidance** | Add diagrams or reference images for exercise form alongside text instructions. |
| **Basic charts & trends** | Visual progress tracking — weekly volume, reps over time, consistency calendar. |
| **Dark mode** | System-preference-aware dark theme for comfortable use in low-light environments. |

---

## Phase 3 — Advanced Features

| Feature | Description |
|---|---|
| **Cloud sync with optional accounts** | Allow users to create an account and sync workout data across devices. Enables backup and device migration. |
| **Audio cues & voice guidance** | Voice countdown for rest timer, verbal prompts for next exercise, "Start set now" alerts without looking at screen. |
| **Custom workout scheduling** | Calendar-based scheduling — plan workouts for specific days, get reminders, track adherence. |
| **Export data (PDF/CSV)** | Download workout history as printable PDF or spreadsheet for personal records or sharing with trainers. |

---

## Future Expansion Possibilities

- **AI-powered plan generation** — User inputs goal (e.g., "build muscle upper body"), duration, and available equipment. App suggests a balanced routine automatically.
- **Voice control** — "Next set", "Skip rest", "I'm done" — hands-free control during workouts using Web Speech API.
- **Wearable integration** — Connect to smartwatches or fitness bands for heart rate monitoring, step count, and calorie burn estimates alongside workout data.
- **Progressive overload tracking** — Analyse history to suggest when to increase weight, reps, or sets based on consistent performance.
- **Multi-user (family) profiles** — Separate profiles on the same device for shared household use without cloud accounts.
- **Routine sharing** — Export/import workout plans via QR code or shareable link — users can share their routines with friends or the community.
- **Exercise library with media** — Browse a library of exercises with images or short video clips demonstrating proper form.
- **Workout templates marketplace** — Community-contributed workout plans discoverable within the app.
- **Gamification** — Streaks, achievements, volume milestones to keep users motivated.

---

*This document is a living idea bank — revisit and prioritise as the app grows and user feedback comes in.*
